# Análise de Dados do Brasil

Repositório dedicado à análise de dados públicos do Brasil usando Python.

## Sobre o Projeto
Este projeto tem como objetivo explorar dados públicos brasileiros,
gerar insights e visualizações que ajudem a entender melhor a realidade do país.

## Tecnologias Usadas
- Python
- Pandas
- Matplotlib
- Google Colab

## Projetos em desenvolvimento
- Análise de emprego e desemprego no Brasil
- Comparativo de dados do ENEM por estado
